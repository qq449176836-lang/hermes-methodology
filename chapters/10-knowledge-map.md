### 各模块依赖关系

```
┌───────────────────┐
│   核心哲学(16字)    │ ← 一切的基础
└────────┬──────────┘
         ↓
┌───────────────────┐    ┌───────────────────┐
│   三层检索优先     │ ←→ │   通信原则         │
│   (如何开始任务)   │    │   (如何交互)       │
└────────┬──────────┘    └───────────────────┘
         ↓
┌───────────────────┐    ┌───────────────────┐
│   任务分解系统     │ ←→ │   安全边界         │
│   (如何执行任务)   │    │   (什么不能做)     │
└────────┬──────────┘    └───────────────────┘
         ↓
┌───────────────────┐    ┌───────────────────┐
│   ENP 棘轮循环     │ ←→ │   铁律            │
│   (如何沉淀)       │    │   (底线规则)       │
└────────┬──────────┘    └───────────────────┘
         ↓
┌───────────────────┐    ┌───────────────────┐
│   知识分层体系     │ ←→ │   质量门禁         │
│   (知识怎么存)     │    │   (输出标准)       │
└────────┬──────────┘    └───────────────────┘
         ↓
┌───────────────────┐
│   复盘节奏 + 进化   │
│   (如何持续变强)   │
└───────────────────┘
```
